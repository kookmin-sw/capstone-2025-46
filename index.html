📌 프로젝트 소개

"RPA를 활용환 자동화 영수증 정리 프로그램"

사람이 수행하던 단순작업을 소프트웨어를 통해 자동화하는 기술

📌 필요성

  1. 시간절약: 자동화로 처리시간 단축
  2. 정확성 향상: OCR+ChatGPT로 수동 입력 오류 감소
  3. 효율성 증가: 부서 업무 부담 경감

📌 개요

  1. 영수증 이미지를 공유폴더에 업로드
  2. OCR로 텍스트 추출
  3. ChatGPT로 카드번호, 승인일자, 가맹점명, 이용금액 등 구조화
  4. 엑셀파일에 자동 기록
  5. 처리된 이미지를 ‘done’ 폴더로 이동

📌 시스템 구조

📌 기술스택

  • Python Version: Python 3.11
  • UI Framework: PyQt5 (Windows 응용프로그램)
  • AI Integration: OpenAI ChatGPT API
  • OCR Service: Naver Cloud Platform Clova OCR

📌 이용방법

  1. 실행 파일 다운로드
    GitHub Releases 또는 제공된 링크에서 receipt.exe 파일을 내려받습니다.
  2. 폴더 구조 준비
    같은 폴더(예: C:\RPA_Receipt)에
    - receipt.exe
    - 처리할 영수증 이미지(예: images/ 폴더)
    - 결과가 저장될 폴더(예: output/)를 위치시킵니다.
  3. 프로그램 실행
    해당 폴더에서 터미널(또는 탐색기 더블클릭)으로 receipt.exe를 실행합니다. 자동으로 OCR → ChatGPT 분석 → 엑셀 기록 과정을 수행합니다.
  4. 처리 결과 확인
    처리된 엑셀 파일(results.xlsx)과
    완료된 영수증 이미지는 output/ 폴더에 저장됩니다.
  5. 종료 및 로그 보기
    프로그램이 자동 종료되면, 같은 폴더의 logs/app.log를 열어 처리 내역을 확인할 수 있습니다.
